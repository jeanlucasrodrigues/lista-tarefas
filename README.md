# 📝Projeto Lista-Tarefas

Este é um aplicativo de desktop moderno desenvolvido em **Python**, focado na organização pessoal e aprendizado. . O projeto utiliza uma interface gráfica estilizada e um banco de dados local para garantir que suas tarefas nunca sejam perdidas.

![Foto do Aplicativo](screenshot.png)

## Funcionalidades

- **Persistência de Dados**: Integração com SQLite para salvar tarefas permanentemente.
- **Interface Moderna**: Desenvolvido com CustomTkinter (suporte a Dark Mode).
- **Segurança**: Janelas de confirmação (Modais) para evitar exclusões acidentais.
- **Organização Dinâmica**: Lista rolável que se adapta ao número de tarefas.

## 🛠️ Tecnologias

- [Python](https://www.python.org/) - Linguagem base.
- [CustomTkinter](https://github.com/TomSchimansky/CustomTkinter) - Interface moderna.
- [SQLite3](https://www.sqlite.org/index.html) - Banco de dados relacional.
